## 项目亮点

- 可随时开启/关闭程序的开机自启动功能。
- 启用 OnEvent 模式，告别使用 while 循环来监听用户操作的方式。
- 在 UI/UE 方面做了很多优化，让自己或别人使用起来更加便捷、舒适。
- 函数式编程。
